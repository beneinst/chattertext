Guida rapida: attivare Chatterbox su un nuovo PC

Questa guida spiega come attivare Chatterbox su un nuovo PC partendo dalla cartella chattertext sincronizzata da GitHub. È pensata per Windows e per l’uso con Mambaforge/Miniforge (o equivalente).

Nota: tutti i comandi vanno eseguiti in Prompt dei comandi (cmd.exe), non in PowerShell.

1. Prerequisiti

  1. Git installato (per clonare/sincronizzare il repository).
  2. Mambaforge / Miniforge / Conda installato (consigliato Mambaforge).
  3. Una GPU NVIDIA (opzionale ma consigliata) se si vuole usare CUDA.
Se la cartella chattertext è già stata clonata/sincronizzata da GitHub, puoi saltare il punto 2.

2. Clonare o aggiornare la cartella chattertext da GitHub

2.1 Clonare per la prima volta

cd D:\Siti GitHub
git clone <URL_DEL_TUO_REPO> chattertext

2.2 Aggiornare una cartella già esistente

cd D:\Siti GitHub\chattertext
git pull

3. Creare l’ambiente chatterbox

Su un nuovo PC, crea un ambiente pulito chiamato chatterbox con Python 3.11:

mamba create -n chatterbox python=3.11

Non è necessario usare mamba activate; useremo sempre mamba run.

4. Installare Chatterbox in modalità editable

Posizionati nella cartella del progetto:

cd D:\Siti GitHub\chattertext

Installa il pacchetto:

mamba run -n chatterbox pip install -e .
Questo comando installa chatterbox-tts e tutte le dipendenze (librosa, diffusers, ecc.) nell’ambiente chatterbox.

5. Installare PyTorch con CUDA (opzionale ma consigliato)

Se il PC ha una GPU NVIDIA e vuoi usare CUDA 11.8:

mamba run -n chatterbox pip install torch==2.6.0+cu118 torchaudio==2.6.0+cu118 --index-url https://download.pytorch.org/whl/cu118

Verifica che la GPU sia visibile:

mamba run -n chatterbox python -c "import torch; print(torch.cuda.is_available(), torch.version.cuda)"
Se il comando stampa qualcosa come True 11.8, la GPU è attiva e CUDA funziona.

6. Disattivare il watermarker (patch necessaria)

Per evitare errori dovuti al modulo PerthImplicitWatermarker, è necessario usare il DummyWatermarker.

  1. Apri il file: 
    src\chatterbox\mtl_tts.py
  2. Cerca la riga (o simile): 
    self.watermarker = perth.PerthImplicitWatermarker()
  3. Sostituiscila con: 
    self.watermarker = perth.DummyWatermarker()
  4. Salva il file.
Questa modifica disattiva il watermark nascosto nell’audio, ma non influisce sulla qualità o sul funzionamento del TTS. Per uso personale e produzione di audiolibri è più che sufficiente.

7. Testare il modello

Per verificare che tutto funzioni, puoi eseguire uno script di test, ad esempio:

mamba run -n chatterbox python chatterbox_auto.py

Oppure, se hai uno script principale diverso, adatta il nome del file:

mamba run -n chatterbox python ChatterText-App-23.py
Se viene generato un file audio senza errori, l’ambiente sul nuovo PC è configurato correttamente.

8. Creare un launcher comodo (facoltativo)

8.1 File batch (.bat)

Crea un file launch_chatterbox.bat nella cartella del progetto con questo contenuto:

@echo off
cd /d D:\Siti GitHub\chattertext
mamba run -n chatterbox pythonw ChatterText-App-23.py

8.2 Launcher invisibile (.vbs)

Crea un file launcher.vbs accanto al .bat:

Set WshShell = CreateObject("WScript.Shell")
WshShell.Run """D:\Siti GitHub\chattertext\launch_chatterbox.bat""", 0, False

Avviando launcher.vbs l’app parte senza mostrare la console.

9. Prompt testuale riassuntivo (per uso futuro)

Se in futuro vuoi chiedere a un assistente di ricostruire questi passaggi, puoi usare un prompt come questo:

Ho una cartella di progetto chiamata "chattertext" sincronizzata da GitHub su Windows.
Voglio creare un nuovo ambiente chiamato "chatterbox" con Python 3.11 usando mamba,
installare il progetto con "pip install -e .", installare torch+torchaudio con CUDA 11.8,
disattivare il PerthImplicitWatermarker usando DummyWatermarker in mtl_tts.py, e creare
un launcher .bat/.vbs che avvii ChatterText-App-23.py senza mostrare la console.
Genera una guida passo passo in italiano, con i comandi esatti da eseguire in cmd.exe.

Questa pagina può essere salvata direttamente nella cartella del progetto (ad esempio come GUIDA_SETUP_CHATTERBOX.html) e aperta in qualsiasi browser sul nuovo PC.
